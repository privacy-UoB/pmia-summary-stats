MicroRNA expressions are real valued numbers that represent how much of the RNA molecule is active in a cell at a given moment in time.
Individuals can contribute their miRNA expression profiles to public datasets to enhance medical research.
Prior work (Backes et al.) has demonstrated that the L1 Distances Difference and Likelihood-Ratio Test can be used to infer membership using complete miRNAs expressions.
This project furthers this work to investigate inference using a subset of the miRNA expressions and on a noisy dataset.
